# lazierlibrarian.koplugin
KOReader plugin for adding books to LazyLibrarian and downloading them

Hacked together from the Z-lib KOReader plugin: https://github.com/OctoNezd/zlibrary.koplugin/

## Configuration
1. Create a file named `ll.conf` in KOReader's root directory (probably at `/mnt/onboard/.adds/koreader`). 
2. The conf file must contain two lines -- URL and KEY
```
URL=https://lazylibrarian.domain.tld
KEY=lazylibrariansapikey
```
